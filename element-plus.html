Element Plus #

一套为开发者、设计师和产品经理准备的基于 Vue 3 的桌面端组件库。

随着 Vue 3 的发布，Element UI 也在同年开始进行升级重构，新项目定名为 Element Plus，并在 2020 年年底发布了第一个预览版，经历了一整年近百个版本后，目前已经发布到 1.2.0-beta.6 版，但始终没有真正的 release 版。不过相信 Element 团队开发出的产品的稳定性禁得住考验。

新组件 #

  • Scrollbar 滚动条
  • Space 间距
  • Affix 固钉

重点：

  • Select V2 虚拟列表选择器（beta）
  • Tree V2 虚拟化树形控件（beta）
